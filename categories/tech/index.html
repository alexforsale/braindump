org-roam

Org-roam is a knowledge management system, or personal wiki, built on top of Emacs’ Org-mode. It implements a networked thought system, similar to Roam Research, allowing users to create interconnected notes in a non-hierarchical way. This facilitates effortless linking and discovery of connections between ideas, effectively creating a “second brain” for managing information. virt-manager

virt-manager virt-manager is a graphical tool, a frontend, for managing virtual machines (VMs) using libvirt. It primarily works with KVM (Kernel-based Virtual Machine) but also supports Xen and LXC containers. It offers features like creating and configuring new VMs, managing existing ones, monitoring performance, and accessing virtual consoles.
