* libinput
Create a /etc/X11/xorg.conf.d/30-touchpad.conf.
#+begin_example
Section "InputClass"
    Identifier "touchpad"
    Driver "libinput"
    MatchIsTouchpad "on"
    Option "Tapping" "on"
    Option "NaturalScrolling" "true"
EndSection
#+end_example
